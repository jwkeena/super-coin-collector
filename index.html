Every game, you will be assigned a random total number of coins.


There are four blocks below. By clicking a block, you will add a specific number of coins to your total.


You win the game by matching the number of coins exactly. You lose the game if your total score goes above the target number.


Every time the game starts, the coin value of each block will change.
